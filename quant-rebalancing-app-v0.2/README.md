# 量化配权软件 v0.2

## 🆕 新增功能

### 1. 最优夏普配权方法
- 新增 **Maximum Sharpe Ratio** 配权方法
- 可自定义无风险利率
- 可选择是否允许做空
- 与等权、风险平价、最小方差并列

### 2. 灵活的交易成本设置
- **买边成本**：可设置百分比或固定金额
- **卖边成本**：可设置百分比或固定金额
- **滑点设置**：交易执行时的价格偏差
- **按资产单独配置**：每个资产可设置不同的成本参数
- 支持全局默认设置

### 3. 三页式验算报告
导出的Excel报告包含3个Sheet：

#### Sheet 1: 仓位时间序列
- 以时间为index
- 各资产仓位（市值）为列变量
- 各资产权重变化

#### Sheet 2: 绩效指标
- 年化收益率、夏普比率、最大回撤等
- Sortino比率、Calmar比率
- 交易次数、总交易成本

#### Sheet 3: 验算页面
- 资产开盘价、收盘价（如有）
- 各资产持仓市值、权重
- 组合净值、日收益率、累计收益率
- 调仓标记（方便定位买卖点）

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

## 使用流程

1. **数据管理** → 上传资产数据（Excel或CSV）
2. **配权计算** → 选择配权方法（含最优夏普）
3. **交易成本设置** → 配置各资产的交易成本
4. **回测分析** → 运行回测，查看绩效
5. **结果导出** → 下载3个Sheet的Excel报告

## 交易成本配置示例

### 股票（A股）
- 买入成本：0.03%（券商佣金，最低5元）
- 卖出成本：0.13%（0.03%佣金 + 0.1%印花税）
- 滑点：0.05%

### ETF
- 买入成本：0.01%（免印花税）
- 卖出成本：0.01%（免印花税）
- 滑点：0.02%

### 期货
- 买入成本：固定手续费（按合约）
- 卖出成本：固定手续费（按合约）
- 滑点：1个tick
