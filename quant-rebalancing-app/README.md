# 量化配权软件

基于Streamlit的量化投资组合配权与回测工具。

## 功能特性

### 📊 数据管理
- 支持多资产数据导入（Excel/CSV）
- 自动数据清洗和日期对齐
- 数据预览和验证

### ⚖️ 配权算法
- **等权配权** (Equal Weight)
- **风险平价** (Risk Parity)
- **最小方差** (Minimum Variance)
- **最大夏普** (Maximum Sharpe Ratio)

### 📈 回测分析
- 历史回测
- 基准对比
- 绩效指标计算
- 收益率曲线可视化
- 回撤分析

### 💾 结果导出
- 导出配权结果（CSV/Excel）
- 导出回测报告

## 安装

### 1. 安装依赖

```bash
cd quant-rebalancing-app
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

## 使用指南

### 第一步：上传数据

1. 在"数据管理"页面上传资产数据文件
2. 每个文件代表一个资产
3. 支持Excel (.xlsx) 和 CSV格式
4. 文件应包含日期列和价格列

### 第二步：配权计算

1. 在"配权计算"页面选择配权方法
2. 选择参与配权的资产
3. 点击"开始计算"
4. 查看权重分配结果

### 第三步：回测分析

1. 在"回测分析"页面选择基准资产
2. 设置再平衡频率和初始资金
3. 点击"开始回测"
4. 查看绩效指标和收益率曲线

### 第四步：导出结果

在"结果导出"页面下载配权结果和回测报告。

## 数据格式要求

### Excel/CSV文件应包含以下列：

- **日期列**: 日期、date、Date、时间等
- **价格列**: 收盘价、close、Close等

### 示例格式：

| 日期 | 收盘价 | 开盘价 | 最高价 | 最低价 | 成交量 |
|------|--------|--------|--------|--------|--------|
| 2024-01-01 | 100.5 | 99.8 | 101.2 | 99.5 | 1000000 |
| 2024-01-02 | 101.3 | 100.5 | 102.0 | 100.2 | 1200000 |

## 算法说明

### 等权配权
所有资产权重相等，最简单的配权方法。

### 风险平价
每个资产对组合总风险的贡献相等。适合追求风险分散的投资组合。

### 最小方差
最小化组合方差，得到风险最低的配置。适合保守型投资者。

### 最大夏普
最大化风险调整后收益（夏普比率）。适合追求效率的投资者。

## 注意事项

1. 数据质量：确保数据完整且准确
2. 样本量：至少需要30个交易日数据
3. 相关性：高度相关资产可能影响优化结果
4. 过拟合：回测结果不代表未来表现
