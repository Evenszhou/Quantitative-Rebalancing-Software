## 💡 技术要点

### BL模型核心公式
```
# 市场均衡收益
Π = λ Σ w_market
其中 λ = (E[R_p] - R_f) / σ_p²

# 后验期望收益
E[R] = [(τΣ)^(-1) + P'Ω^(-1)P]^(-1) [(τΣ)^(-1)Π + P'Ω^(-1)Q]

# 后验协方差
Σ_BL = (1+τ)Σ - [(τΣ)^(-1) + P'Ω^(-1)P]^(-1) P'Ω^(-1)P Σ
```

### 关键参数
- **τ (tau)**: 不确定性参数，通常0.01-0.05
- **Ω (omega)**: 观点不确定性矩阵
- **P**: 观点矩阵
- **Q**: 观点收益向量
